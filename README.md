# Aplikasi Pendataan Pegawai CRUD (V2 - Sistem Tab & Ekspor)

Sistem Informasi Manajemen Pegawai berbasis web ringkas dengan fitur input terklasifikasi, ekspor instan, dan tinjauan menyeluruh data kualifikasi SDM.

## ✨ Pembaruan Fitur Terbaru
1. **Form Input Ter-Klaster (Tabs)**: Pembagian form ke dalam 4 Tab logis (*Data Pribadi, Pekerjaan, Pendidikan, serta Keluarga & Kontak*).
2. **Tabel Ringkas Premium**: Hanya menampilkan data inti yang paling sering dicari.
3. **Modul Detail View**: Akses satu-klik untuk memunculkan modal pop-up yang memuat ke-29 parameter lengkap per individu.
4. **Ekspor Excel**: Download seluruh database lokal ke dalam file `.xlsx` secara instan.
5. **Ekspor PDF**: Cetak berkas laporan tabel ringkas dalam format PDF berorientasi horizontal secara rapi.
